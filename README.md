# Configuration options

Refer to: `bin/start-env.sh`

- `GS_HOME` - path to product directory
- `GS_SERVER_MANAGERS` - host ip address of manager 
 
Refer to: `bin/run.sh`

- -DtestCycles=1000
- -DnumberOfThreads=1
 
# How to run?
Refer to: `bin` directory

1. ./start-managers.sh &
2. ./start-gscs.sh
3. ./start-partitions.sh
4. ./run.sh
